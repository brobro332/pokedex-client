### ✅ **개요**

- 포켓몬스터 도감을 확인할 수 있는 클라이언트 애플리케이션입니다.

### ⏱ **개발 기간**

- `v1.0.0` (`2025-08-09 ~ 2025-08-10`): 도감 전체 목록 및 상세정보 조회 등 기본 화면 개발

### 🛠 **프로젝트 환경**
- `Language`: `Typescript`
- `Framework`: `Vue.js 3`
- `Open API`: `PokéAPI`
- `IDE`: `Visual Studio Code`
- `UI Framework`: `Tailwind CSS`

### 🎬 **설치 및 실행**

```bash
npm install
npm run dev
```
- 설치 및 실행 후 `http://localhost:5173`에 접속하세요.
